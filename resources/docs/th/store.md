# คำอธิบายแบบเต็ม

เกมที่คุณจะได้ <b>ทุบตี</b>, <b>บดขยี้</b>, <b>ทำลาย</b> และ <b>บี้</b> แมลงที่น่ารำคาญ

มีด่านให้เล่นไม่จำกัดจำนวนครั้ง โดยแมลงจะมาสร้างความรำคาญให้คุณในสวน ในครัว และบนโต๊ะอาหารของคุณ
