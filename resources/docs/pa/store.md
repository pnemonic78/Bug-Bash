# ਪੂਰਾ ਵੇਰਵਾ

ਉਹ ਖੇਡ ਜਿੱਥੇ ਤੁਸੀਂ <b>ਬਾਸ਼</b> ਕਰਦੇ ਹੋ, <b>ਕੁਚਲਦੇ ਹੋ</b>, <b>ਸਮੈਸ਼</b> ਕਰਦੇ ਹੋ, <b>ਸਕਵੈਸ਼</b> ਕਰਦੇ ਹੋ, ਅਤੇ <b>ਕੁਚਲਦੇ</b> ਹੋ।

ਤੁਹਾਡੇ ਬਾਗ ਵਿੱਚ, ਤੁਹਾਡੀ ਰਸੋਈ ਵਿੱਚ, ਅਤੇ ਤੁਹਾਡੇ ਖਾਣੇ ਦੇ ਮੇਜ਼ ਉੱਤੇ ਬੇਅੰਤ ਪੱਧਰ ਹਨ ਜਿੱਥੇ ਕੀੜੇ ਤੁਹਾਨੂੰ ਤੰਗ ਕਰਨ ਲਈ ਆਏ ਹਨ।
