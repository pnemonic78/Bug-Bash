# Përshkrim i plotë

Loja ku ju <b>godisni</b>, <b>thërmoni</b> dhe <b>shtypni</b> insekte bezdisëse.

Ka nivele të pafundme ku insektet kanë ardhur për t'ju bezdisur në kopshtin tuaj, në kuzhinën tuaj
dhe në tavolinën tuaj të darkës.
