# Nome

Esmagamento de Insetos

# Breve descrição

Um jogo divertido para esmagar insetos irritantes.

# Descrição completa

O jogo onde se <b>esmaga</b> e <b>amassa</b> insetos incómodos.

Existem níveis infinitos onde os insetos vieram incomodar-te no teu jardim, na tua cozinha e na
a sua mesa de jantar.
