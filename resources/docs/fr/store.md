# Nom

Écraser les insectes

# Description courte

Un jeu amusant pour écraser les insectes indésirables.

# Description complète

Le jeu où vous <b>pulvérisez</b>, <b>aplatissez</b> et <b>écrasez</b> les insectes agaçants.

Il existe une infinité de niveaux où les insectes sont venus vous importuner dans votre jardin,
votre cuisine et même sur votre table.

La plupart des insectes sont nuisibles:
🐜 🐝 🐛 🦗 🦟 🦂 🐌

Certains insectes sont des amis:
🦋 🐞

Utilisez les bonus pour vaincre les insectes :
🧁 🌼 📜 💝 🏆 👞 ☠ 🏸 ⚡
