# ചുരുക്ക വിവരണം

വൃത്തികെട്ട ബഗുകളെ തുരത്താനുള്ള രസകരമായ ഗെയിം.

# പൂർണ്ണ വിവരണം

നിങ്ങൾ <b>തട്ടുക</b>, <b>തകർക്കുക</b>, <b>സ്ക്വാഷ്</b>, ശല്യപ്പെടുത്തുന്ന ബഗുകളെ <b>ചതയ്ക്കുക</b>
ചെയ്യുന്ന ഗെയിം.

നിങ്ങളുടെ പൂന്തോട്ടത്തിലും അടുക്കളയിലും അത്താഴ മേശയിലും നിങ്ങളെ ശല്യപ്പെടുത്താൻ ബഗുകൾ വന്ന അനന്തമായ
തലങ്ങളുണ്ട്.
