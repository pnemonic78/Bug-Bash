# சுருக்கமான விளக்கம்

மோசமான பூச்சிகளை அழிப்பதற்கான வேடிக்கையான விளையாட்டு.

# முழு விளக்கம்

தொல்லை தரும் பூச்சிகளை நீங்கள் <b>அடித்தல்</b>, <b>ஸ்குவாஷ்</b> மற்றும் <b>நசுக்குதல்</b> செய்யும் விளையாட்டு.

உங்கள் தோட்டத்திலும், உங்கள் சமையலறையிலும், உங்கள் இரவு உணவு மேசையிலும் உங்களைத் தொந்தரவு செய்ய
எண்ணற்ற நிலைகளில் பூச்சிகள் வந்துள்ளன.
